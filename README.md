# 💈 Barbearia

Este é um site institucional desenvolvido para apresentar os serviços de uma barbearia tradicional com estilo moderno. O projeto foi construído utilizando apenas **HTML5**, **CSS3** e **JavaScript**, com foco em design responsivo e navegação fluida.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3 (com transições, animações e responsividade)
- JavaScript (para navegação e interações simples)
- Google Fonts (Fonte Lato)
- Iconify (ícones estilizados)

---

## 💡 Funcionalidades

- Menu lateral animado tipo hambúrguer
- Banner com mensagem principal
- Seção "Sobre" (história e conceito)
- Lista de serviços com ícones e descrições
- Depoimentos estilizados
- Galeria de imagens
- Agendamento com botão de ação
- Formulário de contato
- Mapa de localização (Google Maps embed)
- Rodapé com redes sociais

---

## 🖥️ Como visualizar o site

1. Clone o repositório ou baixe o `.zip` deste projeto.
2. Extraia o conteúdo em uma pasta local.
3. Abra o arquivo `index.html` diretamente no navegador.
